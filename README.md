# AI Emotion Detection System 🤖💬

## 📌 Project Overview

This project is a Machine Learning based web application that detects human emotions from text and suggests suitable tasks.

## 🚀 Features

* Text Emotion Detection using NLP
* Machine Learning Model (Naive Bayes)
* Interactive Web Dashboard using Streamlit
* Real-time emotion prediction

## 🛠️ Technologies Used

* Python
* Machine Learning
* NLP (TF-IDF Vectorization)
* Streamlit
* Pandas & Scikit-Learn

## 📊 How It Works

1. User enters a sentence
2. Text is converted into numerical form using TF-IDF
3. Trained ML model predicts emotion
4. System displays detected emotion

## 📷 Output Example

Type: "I feel very happy today"
Result: "Happy"
